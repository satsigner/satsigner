import { Callout } from 'fumadocs-ui/components/callout'

To run **satsigner** mobile app, navigate to the `mobile` app folder

```bash
cd apps/mobile
```

If you running on an Android, run:

```bash
yarn android
```

For iOS, run:

```bash
yarn ios
```

<Callout>The first build might take some time</Callout>
